  • 失误4
    执行命令: hexo d 报错没有rsa ssh key
    那么，就先查找有没有ssh key

如果提示：No such file or directory,就是木有ssh key啦。
那就生成一份ssh key
enter passphrase 就是给ssh key设置密码

生成ssh key之后，找到并打开ssh key pub 文件，复制全部内容到github设置里new ssh。
